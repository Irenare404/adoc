# AusDungeon Documentation

This directory is a standalone GitHub Pages site. Its repository is independent from the AusDungeon plugin repository.

## Deploy

Push this repository's `main` branch, then set GitHub repository **Settings > Pages > Source** to **GitHub Actions**.

The custom domain is configured in `CNAME`:

```text
adoc.thispy.top
```

For DNS, add a CNAME record from `adoc` to `Irenare404.github.io`.
